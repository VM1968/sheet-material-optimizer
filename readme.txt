1. Укажите параметры листа
Длина листа - главное измерение в мм
Ширина листа - боковое измерение в мм
Стоимость листа - цена одного листа в рублях
Припуск при резке - допуск на толщину режущего инструмента (обычно 2-3 мм)
2. Добавьте детали для раскроя
Укажите размеры (длина × ширина в мм)
Укажите количество необходимых деталей
Нажмите "Добавить деталь"
3. Нажмите "Оптимизировать раскрой"
Приложение рассчитает оптимальные схемы
Отобразит количество нужных листов
Покажет визуальные схемы раскроя
Рассчитает стоимость и статистику отходов
📊 Пример использования
Исходные данные:

Лист: 2500 × 1250 мм
Детали: 600×300 мм (5 шт), 400×400 мм (3 шт)
Припуск: 2 мм
Цена листа: 1000 руб
Результат:

Листов нужно: 2
Стоимость материала: 2000 руб
Использовано: 84.5% материала
Отходы: 15.5%

Приложение использует модифицированный алгоритм Guillotine (разрезания гильотиной):

Сортировка деталей - детали сортируются по площади (большие сначала)
Размещение строк - детали размещаются построчно с попыткой заполнить строку полностью
Проверка ориентации - для каждой детали проверяется оптимальная ориентация
Учет припусков - между деталями оставляется припуск для режущего инструмента
Минимизация листов - алгоритм стремится использовать минимальное количество листов
📝 Особенности
Детали автоматически ротируются для лучшего размещения
Поддерживает произвольные размеры листов и деталей
Учитывает припуск на толщину режущего инструмента
Визуализация с использованием Canvas для каждой схемы раскроя
Расчет точной стоимости материала
Статистика использования и отходов
🐛 Известные ограничения
Максимум ~50 деталей для оптимальной производительности
Алгоритм не учитывает направление волокна материала
Не предусмотрена поддержка вложенных деталей разных уровней
