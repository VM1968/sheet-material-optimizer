# CHANGELOG

## [2.0.0] - 2026-06-09

### Added
- ✨ **Управление несколькими листами** - теперь можно работать с листами разных размеров и типов материала
- ✨ **Библиотека стандартных размеров** - быстрый доступ к популярным размерам (фанера, ДСП, МДФ, металл, пластик, стекло)
- ✨ **Контроль поворота деталей** - возможность зафиксировать ориентацию детали или позволить алгоритму выбрать оптимальную
- ✨ **Печать** - вывод готовых схем раскроя в красивом формате с полной статистикой
- ✨ **Улучшенный алгоритм оптимизации** - более эффективное размещение деталей с поддержкой ротации

### Changed
- 🔄 Переработан пользовательский интерфейс для работы с несколькими листами
- 🔄 Обновлены стили для лучшей мобильной поддержки
- 🔄 Улучшена логика расчета использования материала
- 🔄 Переработан JavaScript код для лучшей организации

### Fixed
- 🐛 Исправлены проблемы с расчетом площади материала
- 🐛 Улучшена обработка ошибок при оптимизации

### Improved
- ⚡ Более быстрая визуализация схем на canvas
- ⚡ Лучшая обработка больших наборов данных
- ⚡ Отзывчивый дизайн на всех устройствах
